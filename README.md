An interactive AI-powered mock interview system that supports:
- **Technical**, **HR**, and **Managerial** interviews

---

## Features

###  1. Interview Type Selection  
Choose:
- Technical (TR)
- Managerial (MR)
- HR

###  2. Technical Topics  
TR supports topic selection such as:
- Programming  
- Data Structures  
- Algorithms  
- DBMS  
- OS  
- CN  
- OOP  
- Software Engineering  

### 3. Voice & Text Input  
- Answer questions by **typing** or **speaking**  
- Speech recorded until you click **Stop Recording**  
- User‚Äôs spoken text appears in the chat

###  4. AI-Generated Questions  
- Uses **Google Gemini API**  
- Asks topic-based questions **one by one**  
- Maintains history session using Flask + JS

### 5. Live Feedback  
After every answer, AI evaluates:
- Content quality  
- Clarity  
- Tone & modulation  
- Confidence level  
- Score out of 10  

Shown neatly outside the chat box.

###  6. End Interview  
Shows:
- Overall feedback  
- Strengths  
- Weak points  
- Suggestions for improvement
